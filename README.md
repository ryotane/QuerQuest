# QueryQuest Project 046

This is a local AI runtime project aimed at building a stable, long-running environment for AI model execution.

## Structure

- src/: source code
- tests/: test suite
- docs/: documentation
- data/: data storage

## Getting Started

1. Create a virtual environment:
   ```bash
   python3 -m venv .venv
   source .venv/bin/activate
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Run the application:
   ```bash
   python src/main.py
   ```

4. Run tests:
   ```bash
   python -m pytest tests/
   ```

## Development

- Use `develop` branch for feature development
- Merge changes to `main` branch after testing
- Update documentation as needed
