# Project_041: 過去を振り返る能力と長期記憶

## 目的
「過去を振り返る能力」と「長期記憶」を実現し、チャット間で記憶が継承される仕組みを確立する。

## 課題
1. **記憶の継承不全**: チャット間で記憶がリセットされる
2. **過去振り返り機能の欠如**: 過去のプロジェクト履歴を自動的に検索・表示する機能がない
3. **長期記憶の強化**: 重要度の高い記憶を自動的に長期記憶へ保存する仕組みがない

## 実装計画

### Phase 1: 過去振り返り機能 (P0)
- **機能**: 過去のプロジェクト履歴を自動的に検索・表示
- **実装**:
  - `ProjectHistoryStore` を使って過去のプロジェクト履歴を検索
  - 関連するプロジェクトを自動的に表示
  - 重要な決定事項、問題点、解決策、教訓を抽出

### Phase 2: 長期記憶の強化 (P1)
- **機能**: 重要度の高い記憶を自動的に長期記憶へ保存
- **実装**:
  - `MemoryOS` の `consolidate()` を強化
  - 重要度の高い記憶を自動的に長期記憶へ移動
  - 記憶の圧縮機能を実装

### Phase 3: 記憶継承の確立 (P1)
- **機能**: 次チャットでも自動的に記憶が読み込まれる仕組み
- **実装**:
  - `project_041_context.md` を記憶継承の中心ファイルにする
  - 次チャットで自動的にこのファイルを読み込む
  - セッション履歴の保存・復元を確立

## 成功条件
- [ ] 過去のプロジェクト履歴を自動的に検索・表示できる
- [ ] 重要度の高い記憶を自動的に長期記憶へ保存できる
- [ ] 次チャットでも自動的に記憶が読み込まれる

## 注意事項
- 既存の記憶システム（MemoryOS, ProjectHistoryStore, SessionLogger, SessionRestorer）を拡張する
- 過剰設計しない
- 既存のスタイルを維持する
