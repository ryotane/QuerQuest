## 3. Feeling Scale Definitions

### fatigue (0-5)
- 0: Runtime が軽快に動作している
- 1: 少し重く感じるが問題ない
- 2: 明らかに重い
- 3: 操作にストレスを感じる
- 4: 非常に重い、使用を躊躇する
- 5: ほぼ使用できない

### interruption (0-5)
- 0: 全く邪魔ではない
- 1: 少し気になるが許容範囲
- 2: 頻繁な中断がある
- 3: 集中が妨げられる
- 4: 非常に邪魔、作業の妨げになる
- 5: 使用を躊躇するレベル

### calmness (0-5)
- 0: 全く静かではない
- 1: 少し気になるが許容範囲
- 2: 頻繁な動作がある
- 3: 常に何か動いている感じ
- 4: 非常にうるさい、落ち着かない
- 5: 完全に静か

### placability (0-5)
- 0: 放置できない
- 1: 少し不安だが許容範囲
- 2: 数時間なら大丈夫
- 3: 一日なら大丈夫
- 4: 数日でも大丈夫
- 5: 完全に放置可能

### restart_anxiety (0-5)
- 0: 全く不安でない
- 1: 少し気になるが許容範囲
- 2: 復帰に不安がある
- 3: 状態消失の不安がある
- 4: 非常に不安、再開を躊躇する
- 5: 再開できないレベル

### continuation_naturalness (0-5)
- 0: 全く不自然
- 1: 少し不自然だが許容範囲
- 2: 強制されている感じがある
- 3: 中断からの復帰が不自然
- 4: 非常に不自然、再開を躊躇する
- 5: 完全に自然

## 4. Logging Frequency

- Daily: 1回（夜間の終了時）
- Weekly: 週次レポート（感覚ベースの要約）
- Monthly: 月次レポート（哲学の維持確認）
